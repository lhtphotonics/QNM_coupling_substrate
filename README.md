# QNM_coupling_substrate

# Introduction of the data underlying the results in reference [1]
Firstly, unpack the compressed file “Fig.7z” to be the folder “Fig”, where all relevant files are placed. Then, by starting the MATLAB software and running the .m files in Table 1 of the file “Read me.pdf”, the corresponding figures in reference [1] can be generated. The .mat data files underlying the results in reference [1] are loaded by the .m files to generate the figures.

# Citing the data
